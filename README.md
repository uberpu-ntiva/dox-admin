# dox-admin: The Pact Platform Nerve Center

## Overview

`dox-admin` is the central coordination hub for the Pact Platform ecosystem. It houses governance standards, active state memory, and the Supervisor Dashboard.

## Quick Links

*   **[AGENTS.md](AGENTS.md)**: The primary directive and protocol for all agents.
*   **[Supervisor Dashboard](dashboard/public/index.html)**: Visual overview of the system status (locally viewable).
*   **[Governance Standards](governance/standards/)**: The laws of the ecosystem.
*   **[Proposed Restructure](PROPOSED_RESTRUCTURE.md)**: Details on the 2.0 organization.

## Directory Structure

*   `governance/`: Standards, templates, and references.
*   `state/`: Active memory banks and service registry.
*   `dashboard/`: Source for the visual interface.
*   `workspace/`: Worker (Jules) execution area.
*   `archive/`: Historical records.
